# Docs — Mintlify Documentation Starter Kit

A documentation site built with [Mintlify](https://mintlify.com), covering guides for setup, customization, content writing, AI tool integration, and API references.

## Features

- Getting started guides and quickstart walkthrough
- Site customization (settings, navigation)
- Content authoring with MDX, code samples, images, and reusable snippets
- AI tools integration guides (Cursor, Claude Code, Windsurf)
- API reference documentation with endpoint examples

## Tech Stack

- **Mintlify** — Documentation framework
- **MDX** — Markdown with JSX components

## Getting Started

### Prerequisites

- Node.js 18+
- [Mintlify CLI](https://www.npmjs.com/package/mint)

### Local Development

```bash
npm i -g mint
mint dev
```

Preview at `http://localhost:3000`.

### Publishing

Install the [Mintlify GitHub app](https://dashboard.mintlify.com/settings/organization/github-app) to auto-deploy changes on push to the default branch.
